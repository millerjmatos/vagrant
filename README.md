# Templates

> A collection of ready-to-use templates just copy-paste examples.

## Features

- Copy-paste ready
- No setup required — just grab and run
- Continuously updated

## Usage

1. Browse the repository folders
2. Find the template you need
3. Copy the content
4. Paste and adapt to your use case
5. Run it

```bash
# Clone the repository
git clone https://github.com/millerjmatos/vagrant.git

# Navigate to the desired system
cd vagrant/<template>
```

## Prerequisites

1. Git installed on your machine
2. [Vagrant](https://developer.hashicorp.com/vagrant/install) - required for VM-based templates
3. Basic familiarity with the tool/language
